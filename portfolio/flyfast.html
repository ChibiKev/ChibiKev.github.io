FlyFast

Project Information

  • Category: Web
  • Project Date: February 2022 - April 2022
  • Project URL: https://github.com/Aternity/FlyFast

FlyFast is a web application utilizing React (JavaScript) as the front-end and Tornado (Python) as the back-end, designed as a user-friendly flight booking search application.

It empowers users to efficiently search for flights to their desired destinations within specific time frames and make selections.

The primary motivation behind this project is to provide an exemplary application that showcases the utilization of OpenTelemetry data for harnessing and transmitting it to an OpenTelemetry collector. This also enables us to efficiently visualize and analyze the data being sent.
